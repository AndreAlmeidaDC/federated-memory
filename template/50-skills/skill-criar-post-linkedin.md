# Skill: criar-post-linkedin

Trigger: quando pedido para criar um post no LinkedIn

Passos:
1. Leia o Context Pack em /60-context-packs/linkedin-writing.md
2. Identifique o tema central em uma frase
3. Escreva o hook (primeira linha) primeiro — teste 3 variações
4. Escolha o hook mais direto e provocativo
5. Desenvolva o argumento em 3 blocos
6. Feche com uma afirmação, nunca com pergunta
7. Revise: remova todo travessão, todo elogio desnecessário
8. Confirme: entre 800 e 1500 caracteres

Restrições:
- Não use hashtags no final a menos que explicitamente pedido
- Não use linguagem corporativa
- Não use "Em resumo" ou "Concluindo"
- Não use travessões (—) sob nenhuma circunstância
