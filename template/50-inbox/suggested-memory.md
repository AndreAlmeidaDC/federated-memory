# 50-inbox — Sugestões de Memória Pendentes

Este arquivo contém sugestões de memória geradas por agentes aguardando revisão humana.

Nenhuma entrada aqui é memória permanente até você revisar e mover para a pasta correta.

---

## Como revisar

Para cada sugestão:
- **Aprovar:** mova o conteúdo para a nota correta no domínio ou projeto
- **Editar:** ajuste antes de mover
- **Rejeitar:** delete a entrada

Após revisar, apague a entrada deste arquivo e faça commit.

---

## Formato de entrada (gerado pelo agente)

```
## [DATA] — [DOMÍNIO]
**Sugestão:** [o que o agente quer registrar]
**Fonte:** [de onde veio essa informação]
**Confiança:** [alta / média / baixa]
**Destino sugerido:** [caminho da nota onde deveria ir]
```
